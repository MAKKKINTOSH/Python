План для создания бота

====================================================
Телеграм-бот для дедлайнов лабораторных, д/з и т.д.
====================================================

1. Бот должен включать в себя базу данных, в которых будут лежать дедлайны
2. Бот должен будет включать в себя возможность редактирования дедлайнов привелегированными пользователями
3. Бот должен работать автономно (выгрузить на сервер)
4. Нужна поддержка редактирования/удаления дат
5. Использование библиотеки telebot

Основные средства для разработки(планируемые):
СУБД - SQLite 3.0;
Язык - python
